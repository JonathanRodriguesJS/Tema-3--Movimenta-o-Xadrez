README
Este programa em C simula os movimentos de peças de um jogo de xadrez. As peças são a Torre, o Bispo, a Rainha e o Cavalo. A seguir, está o guia para entender como o código funciona e como utilizá-lo.

Descrição das Funções
moverTorre(int num)
Simula o movimento da Torre. A Torre move-se apenas na direção horizontal (direita). A função moverTorre move a Torre num vezes para a direita.


moverBispo(int num)
Simula o movimento do Bispo. O Bispo se move na diagonal para cima e à direita. A função moverBispo move o Bispo num vezes para cima e à direita.


moverRainha(int num)
Simula o movimento da Rainha. A Rainha pode se mover tanto na horizontal, diagonal quanto na vertical. A função moverRainha move a Rainha num vezes para a esquerda.


moverCavalo(int num)
Simula o movimento do Cavalo. O Cavalo se move em um padrão "L". A função moverCavalo move o Cavalo de acordo com o número de movimentos especificados.
